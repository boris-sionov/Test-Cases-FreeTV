# Verify Top Shelf Playback for Live Channels on Main Page

### Description:
Validate that when focusing on a live channel on the Main page, the Top Shelf starts playback smoothly and displays correct channel information.

### Steps:
1. Open the app.  
2. Navigate to the Main page.  
3. Focus on any live channel.  
4. Observe the Top Shelf playback area.  

### Verifications:
* The selected live channel appears in the Top Shelf.  
* Playback starts without issues.  
* Video and audio play smoothly without interruptions or delays.  
* The Top Shelf displays relevant information (e.g., channel name, program details).  

### Expected Result:
* The selected live channel plays smoothly in the Top Shelf.
